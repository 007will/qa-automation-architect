# QA Automation Architect

## Descrição Executiva
Especialista em automação de testes com foco em soluções Cypress para validação de APIs e interfaces. Entrega frameworks confiáveis de execução contínua e relatórios estruturados para times de QA.

## Foco Técnico
- Tecnologias principais: JavaScript, Node.js, Cypress
- Frameworks/Tools: Cypress 14.x, mochawesome, Allure, cypress-plugin-api, cypress-xpath, cypress-file-upload
- Padrões e abordagens: automação de testes API + UI, integração CI/CD, testes smoke, arquitetura de testes e manutenção

## Hard Skills
- Automação Cypress: nível avançado
  - Implementação de suites e2e, testes de API e front/backoffice com comandos personalizados
- Relatórios e métricas de qualidade: nível avançado
  - Integração de mochawesome e Allure, merge de relatórios e geração de dashboard de execução
- Estrutura de testes escalável: nível avançado
  - Organização de scripts de execução segmentada e validação de ambiente
- Qualidade de código de automação: nível avançado
  - Uso de padrões de escrita, modularização e linting específico para Cypress

## Soft Skills
- Organização técnica: demonstrada por documentação de processos de teste e manutenção
- Foco em entrega: validação de correções com automação e regressão contínua
- Comunicação com times técnicos: apresentação clara de resultados, relatórios e prioridades

## Repositórios Representativos
- `automation-qa-suite`: suite de automação Cypress para fluxos críticos de plataforma B2B
- `test-reporting-framework`: framework de relatórios e análise de qualidade de testes
- `ci-quality-drivers`: scripts de execução de testes segmentados para pipeline e validação contínua
